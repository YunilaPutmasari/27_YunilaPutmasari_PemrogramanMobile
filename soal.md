2.  Mengapa sangat penting untuk memahami bahasa pemrograman Dart sebelum kita menggunakan framework Flutter ? Jelaskan!
jawab :
Memahami bahasa Dart sangat penting sebelum menggunakan Flutter karena Flutter dibangun di atas Dart. Jika kita sudah menguasai Dart, kita akan lebih mudah memahami cara kerja dan sintaks yang digunakan dalam Flutter.

3. Rangkumlah materi dari codelab ini menjadi poin-poin penting yang dapat Anda gunakan untuk membantu proses pengembangan aplikasi mobile menggunakan framework Flutter.
Productive Tooling:
Dart menyediakan alat untuk menganalisis kode, plugin IDE, dan ekosistem paket yang besar, membantu meningkatkan produktivitas pengembangan.

Garbage Collection:
Dart menggunakan garbage collector untuk menangani de-alokasi memori, memastikan bahwa memori yang tidak lagi digunakan dapat dibebaskan secara otomatis.
Type Annotations dan Statically Typed:
Type annotations di Dart bersifat opsional tetapi memberikan keamanan dan konsistensi data. Dart tetap aman dengan type inference dan type safety, yang membantu menemukan bug pada saat kompilasi.

Portability:
Dart dapat dikompilasi menjadi kode JavaScript untuk web, serta kode native untuk ARM dan x86, membuatnya fleksibel untuk berbagai platform.

Perkembangan Dart:
Sejak diluncurkan pada 2011, Dart telah berkembang dengan penekanan awal pada pengembangan web dan kini berfokus pada mobile development, khususnya dengan Flutter. Dart 2.0 memperkenalkan perubahan besar untuk meningkatkan performa dan fleksibilitas.

Eksekusi Kode Dart:
Kode Dart dapat dieksekusi menggunakan Dart VM atau dikompilasi menjadi JavaScript. Dart VM mendukung kompilasi Just-In-Time (JIT) untuk pengembangan dan debugging, serta kompilasi Ahead-Of-Time (AOT) untuk performa tinggi pada aplikasi produksi.

Struktur Bahasa Dart:
Dart menggunakan sintaks yang mirip dengan bahasa pemrograman lain seperti C dan JavaScript. Bahasa ini berbasis object-oriented (OO) dengan fitur seperti encapsulation, inheritance, dan polymorphism.

Operator Dart:
Dart mendukung berbagai operator aritmatika, perbandingan, dan logika. Operator ini dapat digunakan dengan tipe data dan juga dapat diganti sesuai kebutuhan melalui metode dalam kelas.

Fungsi dan Method:
Dart menggunakan fungsi (function) dan metode (method) untuk memisahkan dan mengorganisir kode. Fungsi di luar kelas, sementara metode adalah bagian dari kelas dan memiliki akses ke instance kelas melalui this.

DartPad:
DartPad adalah alat online untuk menulis dan menjalankan kode Dart dengan cepat. Ini berguna untuk eksperimen dan pembelajaran dasar bahasa Dart.
